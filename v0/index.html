Web Components v0 Demos

basic-bucket-es5: minimal v0 example using ES5

The basic-bucket-es5 component is a basic demo implementation using web components v0 and ES5. It doesn't display all v0 capabilities, it is a minimal implementation just to show how ES5 could be used. For examples of the other capabilities of v0, see the other components implemented using ES2015 (which require less boilerplate, but could be backported to ES5).
