custom-elements-basic-example-es5: minimal Web Components v0 example using ES5

The custom-elements-basic-example-es5 component is a basic demo implementation using web components v0 and ES5. It doesn't display all v0 capabilities, it is a minimal implementation just to show how ES5 could be used. For examples of the other capabilities of v0, see the other components implemented using ES2015 (which require less boilerplate, but could be backported to ES5).

extends-example: example of Web Components v0 extending a (browser) native input component

The extends-example component is a demo implementation using web components v0 and ES2015 that showcases how can you extend a (browser) native input component. The main difference (from others that don't extend browser components) is that you have to extend a different prototype, define the extends property and you must use is="my-tag-name" instead of <my-tag-name>.
