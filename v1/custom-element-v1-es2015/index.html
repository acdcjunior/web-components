custom-element-v1-es2015: minimal Custom Element v1 example using ES2015 (ES6)

The custom-element-v1-es2015 component is a basic demo implementation using (web components) custom elements v1 and ES2015 (ES6). It doesn't display all v1 capabilities, it is a minimal implementation just to show how to get the basic done using ES2015.
